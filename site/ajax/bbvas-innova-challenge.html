Project Description
Last year, Spain’s regional bank BBVA hosted a contest inviting developers to create applications, services and content based on anonymous card transaction data. The first prize went, for instance, an application called Qkly, which helps users plan their time by estimating what time of day a given place will be most overcrowded so as to avoid lines. THere are a few different challenges, one of hich is the Innova Challenge. The Innova Challenge MX is a challenge to find the most creative approaches to transforming BBVA Bancomer´s data into value, focusing on how this data can be enriched when combined with external data sources The Datathon API allows the access to a set of statistics by geographical area (average payment, volume of transactions, number of cards...) based on the aggregation of transactions made by BBVA Bancomer cards.
